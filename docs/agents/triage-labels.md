# Triage labels

| Matt 角色         | GitHub label      | 含义                     |
| ----------------- | ----------------- | ------------------------ |
| `needs-triage`    | `needs-triage`    | 等待维护者评估           |
| `needs-info`      | `needs-info`      | 等待报告者补充信息       |
| `ready-for-agent` | `ready-for-agent` | 已充分定义，可交给 Agent |
| `ready-for-human` | `ready-for-human` | 需要人工实现或判断       |
| `wontfix`         | `wontfix`         | 不计划处理               |

Skill 提到 triage 角色时，使用右侧对应的 GitHub label。修改现有标签体系时，更新本文件右侧列即可。
